## Endpoint

```
POST https://api.digitzs.com/merchants
```

## Overview

Use this endpoint to create a new merchant account. This comprehensive onboarding process collects personal, business, and banking information required for payment processing.

<Note>
  Merchant accounts are created in real-time and can begin processing payments immediately upon approval.
</Note>

## Authentication

| Header | Value | Required |
|--------|-------|----------|
| `x-api-key` | Your API key | Yes |
| `Authorization` | `Bearer {appToken}` | Yes |
| `appId` | Your application ID | Yes |
| `Content-Type` | `application/json` | Yes |

```python Python
import requests

def create_merchant(merchant_data):
    url = "https://api.digitzs.com/merchants"

    headers = {
        "x-api-key": "your-api-key",
        "Authorization": "Bearer your-app-token",
        "appId": "your-app-id",
        "Content-Type": "application/json"
    }

    payload = {
        "data": {
            "type": "merchants",
            "attributes": merchant_data
        }
    }

    response = requests.post(url, headers=headers, json=payload)
    response.raise_for_status()

    merchant_id = response.json()["data"]["id"]
    print(f"New Merchant ID: {merchant_id}")
    return merchant_id
```

```php PHP
<?php

function createMerchant($merchantData) {
    $url = "https://api.digitzs.com/merchants";

    $payload = json_encode([
        "data" => [
            "type" => "merchants",
            "attributes" => $merchantData
        ]
    ]);

    $ch = curl_init($url);
    curl_setopt($ch, CURLOPT_POST, true);
    curl_setopt($ch, CURLOPT_POSTFIELDS, $payload);
    curl_setopt($ch, CURLOPT_HTTPHEADER, [
        "x-api-key: your-api-key",
        "Authorization: Bearer your-app-token",
        "appId: your-app-id",
        "Content-Type: application/json"
    ]);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

    $response = curl_exec($ch);
    $data = json_decode($response, true);
    curl_close($ch);

    return $data["data"]["id"];
}
?>
```

```ruby Ruby
require 'net/http'
require 'json'

def create_merchant(merchant_data)
  uri = URI('https://api.digitzs.com/merchants')
  request = Net::HTTP::Post.new(uri)
  request['x-api-key'] = 'your-api-key'
  request['Authorization'] = 'Bearer your-app-token'
  request['appId'] = 'your-app-id'
  request['Content-Type'] = 'application/json'

  request.body = {
    data: {
      type: 'merchants',
      attributes: merchant_data
    }
  }.to_json

  response = Net::HTTP.start(uri.hostname, uri.port, use_ssl: true) do |http|
    http.request(request)
  end

  data = JSON.parse(response.body)
  data['data']['id']
end
```

## AVS Modes

| Mode | Description |
|------|-------------|
| `Standard` | Requires ZIP code match for card payments |
| `Strict` | Requires full address match for card payments |
| `Disabled` | No AVS verification required |
